La api trata de traer información de usuarios y canchas cargadas en una base de datos. A su vez, se puede crear usuarios o canchas nuevas, editar o eliminar la información existente. También cuenta con una autenticación que no permite realizar ninguna de las acciones mencionadas si el usuario no está logueado.
